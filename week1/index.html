How to Rewire You Brain

The paradigm shift. We need to train our brains
so that we can continue to understand the new technology. Do mobile first.
responsive retrofitting, making something one way to then change it for something else.
if you choose mobile first the downside is its time consuming and costly
for a full rebuild. But it provides the best experince accross many devices.
It addresses the constraints of the mobile size/bandiwdth.
Its future friendly and ready for growth. Desktops aren't as popular anymore if we want
to sell to the demand. Start samll and scale up! Mobile isn't going anywhere.

How to Improve Mobile UX by Understanding Users.

It's really hard to design product by focus groups. A lot of times,
people don't know what they want until you show it to them. -Steve Jobs
Be there. Be useful. Be quick. Immediate. Relevant. Frictionless.
Reachability matters. Speed matters you will need ot make sure your stuff loads quick.
How to make it load fast? Reduce image usage throughout sites/apps.
Use SVG/Webfonts they are lower in bandwidth. Optimize and minify
CSS and JS files, remove unused tags or unused codes so you don't have
to load unnecessary code and remove white space. Use GZIP files to eleminate
unnecessary data.

Better Mobile UX With Strategic Menu Design.

When creating a menu keep in mind about the reachability. Always strive to be amazing!
Menu with a border is the most effective. There are a lot of people who don't know
all the logos. Hit your users right in the happies!

Designing Intuitive and Mobile Friendly Forms

Make it easy. When making forms think of someone who hasn't used this before
and only have questions that you actually need. Enable auto complete on forms.
Auto advance for the user and include the spaces for the card numbers.
Use the proper kayboard for numbers! Save the number as an anchor tag so all they ahve to do is tap!

Designing a Better Mobile Advertising User Experience

We are still using advertisement sizes for desktop, we need to use mobile size.
Pop ups are the worst for mobile. Make it that the ads are small and after being tapped
become larger to read. Stop using pop-ups/pop-overs. Users don't visit your site
for ads only for content. Keep it simple and sleek. Empathy is huge when creating sites.
