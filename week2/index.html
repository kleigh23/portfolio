About "let" and "const"

using let or const to declare variables will ensure they are block scoped,
and it is good practice tp always use them to declare variables.
In general, if a value is unlikelt to change in your code then you should
declare it using const, and let should only be used for storing values
that will need to be reassigned while the program is running.

Pop, Push, Shift, and Unshift

pop() will remove the last item in an array
shift() will remove the first item in the array
push() will append a new value to the end of the array
unshift() appends an item to the beginning of the array.
